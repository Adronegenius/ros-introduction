# ROS Basic
Fork or git clone this repository
## How to build the container
To build the image  
```
.docker/build_image.sh
```
To run the image
```
.docker/run_user.sh
```
You may need to change the owner of the dev_ws, copy the line showing on the terminal
```
sudo chown -R [YOUR USER NAME] /dev_ws
```
Start a terminal
```
terminator
```
